## Common Patterns Across Files

1. **Intervention Score Sheets:** Most files contain sheets with student scores structured as:
   - Row 0: Category headers (quadrants)
   - Row 1: Microcompetency codes (A1, A2, C1, etc.)
   - Row 2+: Student data (Roll Number, Name, Gender, Section, Scores)

2. **Student Identification:** All files use consistent student identifiers:
   - Registration Number (RN) / Roll Number
   - Student Name
   - Gender
   - Section

3. **Microcompetency Codes:** Common codes across files:
   - **A** series: Analytical & Critical Thinking (A1-A5)
   - **C** series: Communication (C1-C5)
   - **E** series: Emotional Intelligence (E1-E3)
   - **P** series: Problem Solving (P1-P4)
   - **T** series: Teamwork (T1-T5)
   - **D** series: Discipline/Fearless (D1-D5)
   - **L** series: Leadership (L1)
   - **N** series: Networking (N1-N4)

4. **Level Structure:**
   - **Level 0:** Persona, Wellness, Behavior, Discipline (Fearless quadrant)
   - **Level 1:** 7-8 interventions (Storytelling, Book Review, etc.)
   - **Level 2:** Review and Reflection interventions (4 Reviews + 4 Reflections)
   - **Level 3:** Advanced interventions (Debate, GD, Case Study, etc.)

5. **Scoring Scale:** Most interventions use a 0-5 scoring scale for microcompetencies.
